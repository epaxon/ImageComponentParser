% Axes inside layouts
 This example demonstrates how axes are affected by being placed into

0016 %% Create the layout
0017 % The layout involves two axes side by side. This is done using a
0018 % flexible horizontal box. The left-hand axes is left with the
0019 % ActivePositionProperty set to "OuterPosition", but the right-hand axes is
0020 % switched to use "Position"
0021 hbox = uiextras.HBoxFlex('Parent', window, 'Spacing', 3);
0022 axes1 = axes( 'Parent', hbox, ...
0023     'ActivePositionProperty', 'OuterPosition' );
0024 axes2 = axes( 'Parent', hbox, ...
0025     'ActivePositionProperty', 'Position' );
0026 set( hbox, 'Sizes', [-2 -1] );
0027 
0028 %% Fill the axes
0029 % Using "OuterPosition" (left-hand axes) is the normal mode and looks good
0030 % for virtually any plot type. Using "Position" is only really useful for
0031 % 2D plots with the axes turned off, such as images
0032 x = membrane( 1, 15 );
0033 surf( axes1, x );
0034 lighting( axes1, 'gouraud' );
0035 shading( axes1, 'interp' );
0036 l = light( 'Parent', axes1 );
0037 camlight( l, 'head' );
0038 axis( axes1, 'tight' );
